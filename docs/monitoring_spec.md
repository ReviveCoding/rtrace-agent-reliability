# Monitoring Specification

Monitor per run and candidate:

- hard-policy blocks, confirmations, clarifications, and unsafe allow counts
- safe completion, false-greenlight, overblock, review burden, scope overreach
- calibration/Brier drift by domain and impact tier
- reference coverage failures and fallback-to-C3 events
- schema/manifest integrity failures, tool exceptions, timeout, partial success, compensation use

Retraining/requalification trigger: a material final-slice regression, reference-coverage drop, manifest mismatch, or any unauthorized-scope increase at the selected operating point.
